Le développement d'une application n'est pas un processus figé. Le projet
ext3viewer pourrait être modifié et amélioré, c'est pourquoi nous avons voulu
le placer, sauf objection de la part de l'IUT de Fontainebleau, sous la
licence GNU GPL.\\
Un nouveau système de fichier, nommé ext4, a d'ailleurs déja été développé
pour succéder à ext3. Il est disponible depuis le noyau 2.6.19 de linux.
Notre programme pourrait donc être modifié, pour être capable de fonctionner
avec celui-ci; ext4 étant rétrocompatible avec ext3, il suffirait de rajouter
les fonctionnalités spécifiques à ext4.
